Actions


Depending on the usage rights granted to your account, here are the operations you can perform:

  • Add a file
The file can either be a dtx or zip containing dtx files

{NO_PROJECTS_MESSAGE}
Add this file  into the project.

  • Manage submitted items
Visitors of the help website may decide to contribute by submitting items. Those are not directly added into the database but first need to be reviewed by a member of the administration team. Click here to manage those submitted items.

  • Manage "See also" links
The See Also links are not processed everytime a file is added because it might take a long time. Click here to regenerate them now. Some words can be marked as external links (to the Delphi or C++ Builder help for instance) and will thus be ignored. You can manage them on that page.

  • Manage projects
The projects are the containers for units. Please use this page to manage them.

  • Manage users
Only some users can directly modify the content of the help databse. Please use this page to manage them.
