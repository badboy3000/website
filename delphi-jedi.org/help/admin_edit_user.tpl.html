Editing a user

{MESSAGE}

Please use the elements below to edit the details of the user.
User Name The name used to login
Password Only set this value if you want to set the password for the user
Full Name
E-Mail
Assigned projects Select all the projects that user has write access to. This value is ignored for administrators.
Can upload Users with this priviledge can upload single DTX files into the help system.
Power user Power users can manage the projects they have write access to. They cannot delete them, but can modify their details, and add files.
Power users are not allowed to manage Users and See Also links.
Admin Admins have full acces to the administration console
