Generelle Einstellungen (Tutorialskonfiguration)

* Größe der Kategorie und Tutorial Logos
- Breite und Höhe geben an wie groß das Logo angezeigt wird.

* Auflistung der Kategorien
- Die Kategorien können ein- und zweispaltig aufgelistet werden.
- Anzahl der aufgelisteten Subkategorien unter der jeweils übergeordneten
  Kategorie (So wie bei den Downloads und Links)

* Bilder-Upload Konfiguration
- User das Uploaden von Bilden erlauben
- Angabe der max. Dateigröße in Bytes
- Angabe der max. Bildbreite in Pixeln
- Angabe der max. Bildhöhe in Pixeln

* Template für Anzeige der Kategorien, Tutorialinfos, Tutorialinhalte
- Eigenes Template für der Anzeige der Kategorien eingeben.
Über 'Help' kann eine Liste der einsetzbaren TAGS aufgerufen werden.
In dieser Liste einfach das entsprechende TAG anklicken und es wird
in das Textfenster übernommen.
Mit aktivieren wird das selbst erstellte Template aktiviert.

* Anzeige der gelinkten Tutorials
- Anzeige im Framebrowser oder in einem neuen Fenster
  Die Anwahl des 'Framebrowsers' hat Vorrang vor den Einstellungen bei
  den Tutorials.
- Anzeige sortieren nach Name, Datum, Klicks oder Bewertungen
- Anzahl der Aufrufe um als populär markiert zu werden
