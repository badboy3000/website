API Conversion Library

Welcome to the Project JEDI API Conversion Library. The API Library is a collection of API header conversions either donated to Proiect JEDI or freely available under the Mozilla Public License (or lesser like BSD). Project JEDI wants to thank all the individuals around the world converting APIs for Delphi - each of them makes Delphi a little bit better!

How can you help?

If you know an API conversion which is missing or a better - maybe more up to date - conversion or a broken link etc. please feel free to report that back to us either using the "Submit API" feature, our Issuetracker, the "Add comment" feature or the "Broken Link" button of this homepage. Your feedback is essential to keep this site one of the best API conversion related resources. One of our APIs isn't up to date? Please feel free to improve it!

Bug reporting

Found a bug? To report a bug please please contact the original authors - especially for API conversions from external sources (e. g. Borlands Code Central) which are not hosted by us - and add the bug to our issuetracker for further reference.

Now, if you want to submit a new API please click here

Project JEDI now wishes you a lot of fun with our API Library and may the source be with you!
