Submitting Header Conversions to Project JEDI
This page will step you through a short procedure to help Project JEDI organize donated files. Please follow the instructions to help ensure that the donation goes smoothly.

All donations must include an info.txt file.  Simply download the info.ini template, fill in the blanks, rename the file to info.txt and include it in your zipfile. For your convenience, a template for your Pascal unit header is included in this file.

It is important to include as much of the requested information as possible. Your original email message will NOT be saved with your submission, so be sure to include any notes in a separate readme.txt file and include it in the pack as instructed in the "How-to" document.

If the zipped conversion pack is smaller than 500Kb, email it to Project JEDI. If it is bigger than 500Kb, do not attach the file but use the email link to request instructions for uploading it to the FTP site.
